Open Survey

把用户反馈直接接入你的仓库工作流

这是一个基于 GitHub Pages、Vercel Serverless Function 与 GitHub Actions 的轻量问卷入口，适合社区收集建议、问题反馈与功能诉求。

  • 前端静态托管： 页面可以直接发布到 GitHub Pages。
  • 服务端隐藏密钥： GitHub Token 只保存在 Vercel 环境变量。
  • 自动统计： 新提交会进入 responses 目录并驱动 Markdown 汇总。

社区用户意见征集

留下你的 GitHub 用户名和建议，我们会把它写入数据仓库并参与自动汇总。

部署前请把页面中的 API 地址替换成你自己的 Vercel 接口地址，格式示例： https://your-vercel-project.vercel.app/api/submit
